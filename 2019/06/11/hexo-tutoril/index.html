站在了大佬Ted Qiu的肩膀上，按照了他的post用Hexo搭建了个人blog，感谢大佬的帮助! Hexo是个专为blog开发的framework, 由md文件的编写来渲染出HTML文本的工具，可以说是节省了大量的开发时间，还是很nice的。网上也有大量的教程可以参考。链接分享:

Node.js安装

首先下载安装最新版本Node.js

如果不确定是否安装过，可以打开terminal输入node -v和npm -v来确定。

Github

作为个程序猿，Github还是必备的, 详细步骤请自行阅读官方文档或者阅读weiyang blog, 里面也有详细的教程, 当完成github的部署之后就可以从GitHub去public个人blog了. 详见Ted的post

安装Hexo

当完成项目文件夹创建之后，打开Terminal，cd到项目目录下，输入npm install hexo-cli-g安装Hexo.

然后输入npm install hexo --save继续安装

通过hexo -v来进行验证.

初始化个人blog，输入hexo init初始化文件夹(注意，目录必须为空，否则会报错)，接着输入npm install安装必备的组件

配置完成之后，输入hexo g生成静态网页，然后输入hexo s打开本地服务器，本地域名默认为：http://localhost:4000/

按ctrl+c关闭本地服务器, 这里给个小建议，在运行本地服务器的时候，可以使用hexo s -debug，当修改并保存文件之后，网页会自动更新，避免了一次次的重启服务器.

Mac系统，如果安装过程成遇到了permission deny问题，请在命令前输入sudo

主题修改

这里使用了Bean Tech主题，Github上clone到本地. 注意当前terminal的根目录.

git clone https://github.com/YenYuHsuan/hexo-theme-beantech ./hexo-beantech

clone完之后需要覆盖本地文件，cp -r hexo-beantech/. ./

再执行npm install,再运行hexo clean && hexo g && hexo s -debug运行本地blog, 效果如图
